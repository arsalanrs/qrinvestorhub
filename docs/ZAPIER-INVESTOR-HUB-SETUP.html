Zapier Setup — Two Zaps

Zap #1 — application submitted (Shape + staff email + welcome email). Zap #2 — portal sign-in link when a customer requests login at /portal. No Supabase Auth emails.

Formspree is NOT used here. IT Support Formspree stays on Intelligence Hub only. Investor Hub uses this webhook + Zapier for all post-submit email and Shape sync.

Zap #1 — Application submitted

Step 1 — Create Catch Hook (Trigger)

  1. Zapier → Create Zap
  2. Trigger: Webhooks by Zapier → Catch Hook
  3. Copy the webhook URL → paste into Vercel as ZAPIER_INVESTOR_SUBMISSION_WEBHOOK_URL
  4. Leave the test tab open — submit a test application from Investor Hub (or use Zapier's "Test trigger" after a real submit)

Optional security filter (recommended)

  1. Add step: Filter by Zapier
  2. Condition: X Questrock Webhook Secret (from headers) Exactly matches your ZAPIER_WEBHOOK_SECRET
  3. Skip this if you don't set the secret env var

Step 2 — Add Paths (3 branches)

After the trigger (and optional filter), add Paths by Zapier. Create three paths — all can run on every submission (no conditions needed), or use zapierPaths if you want to toggle later.

Zap #2 — Portal sign-in link (separate Catch Hook)

Required. Uses its own webhook URL — ZAPIER_PORTAL_LOGIN_WEBHOOK_URL. Fires when a customer requests a sign-in link at /portal.

Step 1 — Create Zap #2

  1. Zapier → Create Zap (new, separate from submission Zap)
  2. Trigger: Webhooks by Zapier → Catch Hook
  3. Copy the new webhook URL → Vercel: ZAPIER_PORTAL_LOGIN_WEBHOOK_URL
  4. Action: Microsoft Outlook → Send Email
Outlook fieldMap from Catch Hook
Toemails customer to
Subjectemails customer subject
Body formatHTML
Bodyemails customer html

Payload always has event = investor_portal.login_requested and a ready-made loginUrl (one-time link, 30 min).

  "email": "borrower@example.com",

    "customer": {
      "to": "borrower@example.com",
      "subject": "Sign in to QuestRock Investor Portal",
      "html": "...",
      "text": "..."
    }
  }
}

Test Zap #2 locally after pasting your Catch Hook URL:

  1. Deploy Investor Hub with webhook URL set

Troubleshooting

Webhook not firing: Check Vercel logs for [submit] Zapier webhook. Confirm ZAPIER_INVESTOR_SUBMISSION_WEBHOOK_URL is set on Production.

Empty HTML in email: In Gmail/Outlook step, choose "HTML" body type and map emails staff html — not the plain text field.

Shape duplicate leads: Add a Shape search step on email/phone before create.

Customer can't log in: Confirm ZAPIER_PORTAL_LOGIN_WEBHOOK_URL is set on Vercel, Zap #2 is ON, and portal_login_tokens migration ran in Supabase.

Double Shape sync: Keep ENABLE_SHAPE_SYNC=false when Zapier owns Shape.
